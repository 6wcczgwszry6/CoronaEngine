## 推荐阅读顺序

1. `overview/ONE_PAGE_OVERVIEW_cn.md`：单页理解项目定位、主链路和成熟度。
2. `overview/QUICK_START_cn.md`：用最短时间建立工程认知。
3. `architecture/ARCHITECTURE_MAP_cn.md`：用图理解系统关系和数据模型。
4. `overview/PROJECT_SUMMARY_cn.md`：查看完整项目概览和阅读路线。
5. `development/DEVELOPER_GUIDE_cn.md`：进入日常开发流程。
